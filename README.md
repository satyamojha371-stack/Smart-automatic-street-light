# Smart Automatic Street Light

## Description
This project uses Arduino, LDR, and IR sensors to control street lights automatically.
Lights turn ON in dim mode at night and become bright when vehicles are detected.

## Components Used
- Arduino Uno
- LDR Sensor
- IR Sensors (4)
- LEDs
- Resistors
- Jumper Wires

## Working
- LDR detects day and night
- At night, lights turn ON in dim mode
- IR sensors detect vehicles
- Lights become bright when vehicle passes
- After that, lights return to dim mode
